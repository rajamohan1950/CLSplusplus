Zero Code Integration

Your app already calls OpenAI or Anthropic. Just change the base URL. Every conversation now has persistent memory. Nothing else changes.

1

Find your LLM call

Wherever you call OpenAI or Anthropic in your code.

2

Change one line

Point the base URL to CLS++. That's the entire integration.

Before (no memory)

client = OpenAI( # default: api.openai.com ) # User chats. Forgets everything next session.

After (persistent memory)

client = OpenAI( base_url="http://localhost:8181/v1" ) # Same code. Now remembers every user forever.
# Python — OpenAI with memory (1 line change) from openai import OpenAI client = OpenAI( api_key="your-openai-key", base_url="http://localhost:8181/v1" # ← This is the only change ) response = client.chat.completions.create( model="gpt-4", messages=[{"role": "user", "content": "Remember, I prefer Python"}] ) # CLS++ intercepts the call, injects memory context, # forwards to real OpenAI, stores the conversation. # Next session, GPT-4 already knows the user prefers Python.
# Python — Anthropic with memory (1 line change) import anthropic client = anthropic.Anthropic( api_key="your-anthropic-key", base_url="http://localhost:8181" # ← This is the only change ) message = client.messages.create( model="claude-sonnet-4-20250514", max_tokens=1024, messages=[{"role": "user", "content": "I just switched to TypeScript"}] ) # Next time, Claude already knows about the TypeScript switch.

Works with any OpenAI-compatible client: LangChain, LlamaIndex, CrewAI, AutoGen — just set base_url.
