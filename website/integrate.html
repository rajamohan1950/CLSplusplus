Chrome Extension

Silent cross-model memory for ChatGPT, Claude, Gemini. Auto-captures conversations. Zero setup.

What It Does

The CLS++ Chrome extension sits invisibly in your browser. When you chat with any AI — ChatGPT, Claude, or Gemini — it captures your preferences, corrections, and facts. Next time you use any model, those memories are automatically injected into the conversation. The AI already knows you.

Quick Integration (30 seconds)

1. Download the extension below
2. Open chrome://extensions → Enable Developer Mode
3. Drag the .zip file onto the page
4. Visit ChatGPT or Claude — memory is active
// That's it. No API key, no config, no code.

Working Example — Try It Now

After installing, go to chatgpt.com and type:

"My name is [your name] and I prefer Python for coding"

Now open claude.ai and ask:

"What's my name and what language do I prefer?"

Claude will know — even though you never told it. That's CLS++ working.

How To Use

✓ Click the 🧠 icon in your toolbar to see memory count
✓ Toggle "Auto-inject" to control when memories are used
✓ Toggle "Local mode" if running CLS++ on localhost
✓ Link your CLS++ account to sync with the dashboard
✓ Memories are per-browser — private to your machine
✓ Works on ChatGPT, Claude, Gemini, and Copilot
