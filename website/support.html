Frequently Asked Questions

How do I install CLS++?

Load the extension in Chrome via chrome://extensions → Developer mode → Load unpacked. Point it to the extension folder. Then start the local server and you're ready to go.

The extension popup says "Server unreachable"

Make sure the CLS++ server is running on port 8080. Check:

  • Open http://localhost:8080/health in your browser — it should return {"status":"ok"}
  • If not running, start it: cd prototype && PYTHONPATH=../src python3 -m uvicorn server:app --host 0.0.0.0 --port 8080
  • Make sure "Local mode" is ON in the extension popup

Memories are not being captured

  • Check that "Auto-inject" is ON in the extension popup
  • Check that "Local mode" is ON if running locally
  • Open the Chrome DevTools console on the AI site — you should see [CLS++] fetch interceptor active on chatgpt/claude/gemini
  • Check http://localhost:8080/ui/memory.html to see if memories appear

How do I delete all my memories?

Delete the file ~/.clspp/memories.json and restart the server. Or use the API: curl -X DELETE http://localhost:8080/api/memories/YOUR_UID

Which AI sites are supported?

  • ChatGPT — chatgpt.com and chat.openai.com
  • Claude — claude.ai
  • Gemini — gemini.google.com

Is my data private?

In local mode (default), all data stays on your machine. Nothing is sent to any external server. See our Privacy Policy for details.

The brain icon doesn't appear in the menu bar

The menu bar daemon needs Accessibility permission. Go to System Settings → Privacy & Security → Accessibility and enable Python or Terminal.
