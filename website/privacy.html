Privacy Policy

Last updated: April 1, 2026

CLS++ Memory ("CLS++", "we", "our") is a browser extension and local memory server that provides cross-model memory for AI chatbots. This privacy policy explains how we handle your data.

Data We Collect

CLS++ stores the following data to provide its memory functionality:

  • Conversation snippets — Short facts extracted from your messages to ChatGPT, Claude, and Gemini (e.g., "My name is Alex", "I work at Acme Corp").
  • User preferences — Settings like auto-inject toggle and local mode toggle, stored in Chrome's local storage.
  • A random user identifier — A randomly generated ID stored locally to associate your memories. This is not linked to your real identity.

Where Data Is Stored

  • Local mode (default) — All data stays on your computer at ~/.clspp/memories.json. Nothing is sent to any external server.
  • Cloud mode — If you enable cloud mode, data is sent to our server hosted on Render (clsplusplus.onrender.com) over HTTPS. Data is stored in server memory and is not persisted to a database.

Data We Do NOT Collect

  • We do not collect your name, email, or any personally identifiable information.
  • We do not track your browsing history.
  • We do not sell or share your data with third parties.
  • We do not use analytics or tracking scripts in the extension.
  • We do not store your full conversation history — only short factual snippets.

How Data Is Used

Your stored memories are used solely to provide context to AI chatbots when you send messages. When you type a message on ChatGPT, Claude, or Gemini, the extension prepends relevant memories so the AI can respond with awareness of your prior conversations.

Data Sharing

We do not share your data with any third parties. Your memories are only sent to the AI chatbot you are actively using (ChatGPT, Claude, or Gemini) as part of your message, so the AI can provide personalized responses.

Data Deletion

You can delete all your memories at any time:

  • Local mode: Delete the file ~/.clspp/memories.json
  • Via the API: Send a DELETE request to /api/memories/{your-uid}
  • Uninstall: Removing the extension stops all data collection immediately.

Permissions

The extension requests these Chrome permissions:

  • storage — To save your preferences (auto-inject, local mode toggle).
  • Host permissions for AI sites — To inject memory context into your messages on ChatGPT, Claude, and Gemini.
  • Host permissions for localhost — To communicate with the local memory server.

Children's Privacy

CLS++ is not directed at children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this policy from time to time. Changes will be posted on this page with an updated date.
