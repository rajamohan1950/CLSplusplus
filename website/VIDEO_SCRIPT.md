# CLS++ Demo Video Script (1 minute)

**Purpose:** Show the multi-model context loss problem and how CLS++ solves it.

**Suggested flow:**

1. **0:00–0:15** — Sarah opens Claude. She types: "Remember: I prefer Python 3.11, use type hints, and our project uses FastAPI." Claude responds. She adds: "Our API base URL is /api/v1." Claude acknowledges.

2. **0:15–0:30** — Sarah switches to ChatGPT (or Gemini). She asks: "What's our API base URL?" ChatGPT: "I don't have that information." She asks again about Python preferences. Same answer. **Context is gone.**

3. **0:30–0:45** — Cut to Sarah with CLS++. She writes the same preferences to CLS++. She switches to ChatGPT. Before each query, CLS++ injects context. She asks the same question. ChatGPT: "Your API base URL is /api/v1." **Context is preserved.**

4. **0:45–1:00** — Quick text overlay: "CLS++. Memory that lives outside the model." Logo. CTA: "Get started at clsplusplus.com"
