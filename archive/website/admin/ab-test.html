How to View Live Data

This dashboard shows placeholder data. For live metrics:

  1. Go to your PostHog dashboard at us.posthog.com
  2. Create an Insight with events: page_viewed, cta_clicked, landing_feedback
  3. Break down by property: variant
  4. Use Funnels to measure: page_viewed → cta_clicked → signup_completed per variant
  5. Force a variant for testing: /?variant=A, /?variant=B, or /?variant=C
