CLS++ Memory

One click opens your browser’s extension page so you can load CLS++. This page uses the local server to find Chrome on disk — not the browser you opened this tab in.

Memory is installed when the menubar app and 127.0.0.1:8080 are running. Use the button below to open Chrome (or Brave / Arc / Edge), turn on Developer mode, then Load unpacked and pick the extension folder.

Manual steps

  1. Open chrome://extensions in Chrome (paste into the address bar).
  2. Enable Developer mode.
  3. Click Load unpacked and select the extension folder next to server.py (or under ~/.clspp/extension after install).
